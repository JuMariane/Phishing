## 🚀 Sobre o Projeto

O **Phishing Scanner** foi desenvolvido para identificar padrões suspeitos em URLs e arquivos, auxiliando usuários e profissionais de segurança a validar a procedência de links antes de qualquer interação.

### 🛠️ Funcionalidades Principais
* 🔍 **Scan de URLs:** Verificação em tempo real contra bases de dados de phishing.
* 📊 **Relatórios Detalhados:** Informações técnicas sobre a ameaça detectada.
* ⚡ **Interface Intuitiva:** Projetado para ser rápido e eficiente.
