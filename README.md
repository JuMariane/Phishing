## 🎯 O Desafio da Feira de Tecnologia

Este projeto nasceu de um desafio acadêmico no curso de **Desenvolvimento de Sistemas**. Recebi o tema **Phishing** e decidi criar uma experiência prática para a feira da turma: um site que simula uma interface confiável para mostrar o quão vulneráveis nossos dados podem estar.

> **"A melhor forma de ensinar alguém a não cair em um golpe é mostrando como ele funciona por dentro."** ✨

### 🎣 A Isca: O "Curso Gratuito"
Para tornar a simulação eficaz, utilizei uma das estratégias de engenharia social mais comuns da atualidade: **a promessa de um curso gratuito**. 
* **O Gatilho:** O site atrai o usuário com a oferta de aprendizado sem custo.
* **A Captura:** Para "garantir a vaga", o usuário insere e-mail e senha. É exatamente aqui que a maioria das pessoas cai, movidas pela urgência e pelo benefício oferecido.
* **A Revelação:** No momento da inscrição, o sistema registra as credenciais e o horário do acesso, exibindo tudo ao final para provar como é fácil ser "pescado" por uma interface bonita.

---

## 🚀 Sobre as Funcionalidades Técnicas

Além da simulação, o core do projeto foca em identificar padrões suspeitos em URLs e arquivos, auxiliando na validação de links suspeitos.

* 🔍 **Scan de URLs:** Verificação em tempo real contra padrões de phishing.
* 📊 **Relatórios Detalhados:** Exibição clara das informações técnicas capturadas.
* ⚡ **Interface Intuitiva:** Projetada para ser convincente e causar impacto visual.
